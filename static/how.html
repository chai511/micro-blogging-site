Micro Blogging Site Design Details

        1.	This is created using HTML, CSS, Flask, MongoDB. 
        2.	Flask provides an easier way to code for Python developers. 
        3.	Flask uses Jinja2 Templates where it does support most of the Python Data Structures as it is.
        4.	MongoDB is used for storing the content of the Blogs so that users can always retrieve the content globally
        5.	Python ofcourse the backend part to develop Flask application
        6.	HTML and CSS are for general purpose web development and UI/UX 
        7.	Deployed using Render.com
